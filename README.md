# chat-ia

Aplicação para entender e aplicar conceitos sobre o funcionamento da integração de uma IA em um chat.

- Foi utilizada a LLM Ollama com a model llama3.2 para gerar as respostas;
- Todas as interações são registradas no banco de dados sqlite;

## Instalação
Após clonar o projeto, execute o seguinte comando no seu terminal:
```bash
make install
```

Se tudo der certo, acesse `http://localhost:8181` para acessar o chat.

## Referências:
- [Ollama LLM](https://github.com/ollama/ollama)
